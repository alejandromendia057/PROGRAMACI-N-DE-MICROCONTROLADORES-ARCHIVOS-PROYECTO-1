MAQUINA DE ESTADOS - RELOJ

MODO_ESTADO [0x00 - 0x03]  |  5 BOTONES  |  3 TIMERS  |  ISR PINCHANGE
ESTADO 0 - Mostrar Hora
ESTADO 1 - Mostrar Fecha
ESTADO 2 - Config. Hora / Alarma
ESTADO 3 - Config. Fecha
BTN5 - Cambiar Estado
RESET / INICIO
SETUP
Prescaler 1MHz  |  Timer0 250ms  |  Timer1 60s
Timer2 2ms (multiplex)  |  PCINT PORTC
Init vars SRAM  |  Alarma=0xFF  |  SEI
RELOJ
Si INDICADOR_TMR1=1 incrementa hora y actualiza displays
FECHA
Si INDICADOR_CAMBIO_DIA=1 incrementa fecha con logica mensual
BTN 1-4
Ejecuta accion segun MODO_ESTADO actual
BTN 5
Cicla MODO_ESTADO 0->1->2->3->0 y actualiza LEDs
ALARMA
Compara hora actual con hora de alarma y activa buzzer
RJMP MAIN_LOOP

// Rutinas de Interrupcion //
ISR_TIMER0
Periodo: 250 ms  |  Prescaler 1024
  • Decrementa CONTADOR_ALARMA
  • Apaga BUZZER si contador llega a 0
  • Cada 2 desbordamientos: Toggle LED PC5
ISR_TIMER1
Periodo: 60 s  |  Prescaler 1024
  • Setea INDICADOR_TMR1 = 1
  • Loop principal detecta y actualiza hora
  • Recarga TCNT1 = 0x1B1F (6943)
ISR_TIMER2
Periodo: 2 ms  |  Prescaler 8
  • Multiplexado de 4 displays (PB0-PB3)
  • Lee MODO_DISPLAY para elegir dato
  • Preserva estado del buzzer (bit7 PORTD)
PINC_ISR  (PC-INTERRUPT PORTC)
Trigger: Flanco de bajada en PC0-PC4  |  Pull-up activo
Logica de deteccion:
Lee PINC  XOR  mascara  AND  estado anterior
Filtra flancos de bajada reales
PC0=BTN1 → INDICADOR_BOTON1=1
PC1=BTN2 → INDICADOR_BOTON2=1
PC2=BTN3 → INDICADOR_BOTON3=1
PC3=BTN4 → INDICADOR_BOTON4=1
PC4=BTN5 → INDICADOR_BOTON5=1
